# Название: Mesto - Интерактивная платформа - 4 ПР 

## Описание
Одностраничный сайт , на котором пользователь может взаимодествовать с контентом.

## Технологии:
- Вёрстка по макету в Figma.
- Файловая структура БЭМ по схеме Nested;
- Flexbox;
- Grid;
- Семантические теги;
- Адаптивная вёрстка
- Резиновая вёрстка
